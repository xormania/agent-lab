# .claude — agent-lab

This directory contains Claude Code helpers for `agent-lab`.

Root policy lives in `../AGENTS.md`. `../CLAUDE.md` delegates to it.

Contents:

- `commands/` — project slash-command prompts for review and planning.
- `settings.local.example.json` — optional conservative local settings example. Review before copying to `.claude/settings.local.json`.

Do not store secrets, real API keys, browser profiles, or private env files here.
